English Version

This application respects and protects the privacy of all users. This Privacy Policy explains how we collect, use, store, and protect user information when using the Facebook Login feature.

1. Information We Collect

When users log in with Facebook and grant permission, this application may access the following information:

  • Public profile (such as name and profile picture)
  • Email address (if the user consents to share it)

We will not access any additional Facebook data without explicit user consent.

2. How We Use Information

The information we obtain will only be used for:

  • User authentication and account linkage
  • Providing services related to account management (e.g., login/session handling, personalization)

We do not sell, rent, or otherwise monetize user information, nor share it with third parties except with user consent or as required by law.

3. Data Storage & Retention

We do not retain Facebook user data longer than required for the functionality provided. If retention is necessary, we will inform users of the scope and duration.

4. Security

We implement reasonable technical and organizational measures to protect data. However, no transmission or storage can be guaranteed 100% secure.

5. Third-Party Services

If third-party services are used (e.g., analytics, hosting, backend providers), they may process data under their own privacy policies. We aim to choose providers who comply with data protection standards.

6. Children

This application is not intended to knowingly collect data from children. If you believe a child has provided personal data, please contact us to have it removed.

7. Updates to This Policy

We may update this privacy policy to reflect changes in our practices. Material changes will be posted on this page with an updated date.
